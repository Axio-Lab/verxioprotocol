# Verxio Protocol

## Getting started

```shell
# Clone the repository
git clone https://github.com/Axio-Lab/verxioprotocol.git
# cd into the directory
cd verxioprotocol
# Install dependencies
pnpm install
```

## Development

Format code:

```shell
pnpm fmt
```

Run tests:

```shell
pnpm test
```

Build the project:

```shell
pnpm build
```

## CI

Run all CI checks:

```shell
pnpm ci
```
